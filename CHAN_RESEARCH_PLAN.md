# 沪深300缠论信号量化研究计划

生成日期：2026-06-15

## 1. 研究背景

本项目的目标不是简单寻找“某个买点必涨”的万能规则，而是把缠论中的结构信号转化为可检验、可复盘、可迭代的量化研究问题。

目前 V2 Core 已经完成沪深300全样本批量分析，核心数据来自 Tushare 真实行情，并基于日线、60分钟、30分钟三个周期生成缠论候选信号，包括一买、二买、三买、一卖、二卖等结构事件。

从当前结果看，缠论信号本身更像“结构状态标签”和“候选事件”，而不是可以直接机械执行的买卖按钮。因此后续研究的重点应当是：

1. 判断不同信号到底代表什么市场含义；
2. 找出信号在哪些市场环境、个股状态、周期组合下更有效；
3. 将稳定有效的统计假设转化为交易规则；
4. 再通过回测、样本外验证和实盘跟踪逐步迭代。

## 2. 当前数据基础

当前批量样本：

- 股票池：沪深300成分股
- 完成数量：300 / 300
- 数据区间：2023-01-01 至 2026-06-12
- 信号统计起点：2024-09-01
- 分析周期：日线、60分钟、30分钟
- 复权方式：前复权
- 批量目录：`data/reports/chan_strategy/v2_batch_20260614_120245`

当前已完成的全样本统计主要来自：

- `batch_summary.csv`：单票综合评分与状态
- `batch_signal_stats.csv`：每只股票、每个周期、每类信号的未来收益统计
- `batch_candidates.csv`：候选信号明细
- `batch_latest_states.csv`：最近多周期结构状态

注意：完整 300 只的事件级市场环境分析仍需要进一步优化脚本后重跑。目前市场环境分组结论应作为待复核结论，而不是最终结论。

## 3. 初步统计发现

### 3.1 裸信号不能直接交易

从沪深300全样本看，各类裸信号未来 10 根 K 线平均收益为小正，但胜率不高：

| 信号 | 未来10根均值 | 胜率 |
|---|---:|---:|
| 一买 | 0.090% | 47.19% |
| 二买 | 0.168% | 47.76% |
| 三买 | 0.267% | 47.25% |
| 一卖 | 0.278% | 47.93% |
| 二卖 | 0.141% | 46.81% |

这说明一买、二买、三买不能直接理解为“出现就买”，一卖、二卖也不能直接理解为“出现就卖”。

### 3.2 信号更像结构标签

未来 20 根 K 线维度下，各类信号的平均收益有所提高：

| 信号 | 未来20根均值 | 胜率 |
|---|---:|---:|
| 一买 | 0.317% | 49.12% |
| 二买 | 0.374% | 48.79% |
| 三买 | 0.529% | 48.26% |
| 一卖 | 0.537% | 48.41% |
| 二卖 | 0.318% | 47.98% |

这提示信号后续存在一定正向漂移，但胜率仍接近随机。更合理的方向是把它们作为结构事件，再叠加环境过滤、个股状态过滤和多周期联立。

### 3.3 周期层级非常重要

未来 20 根 K 线表现相对突出的组合包括：

| 周期 | 信号 | 未来20根均值 | 胜率 |
|---|---|---:|---:|
| 日线 | 一卖 | 2.617% | 53.99% |
| 日线 | 二卖 | 1.414% | 49.51% |
| 日线 | 二买 | 1.284% | 49.59% |
| 日线 | 一买 | 1.064% | 50.22% |
| 60分钟 | 三买 | 0.672% | 48.53% |
| 60分钟 | 一买 | 0.609% | 49.78% |

这里最值得注意的是：日线一卖、二卖之后并没有明显下跌，反而存在正向收益。这说明卖点可能不是简单的清仓信号，而更可能是趋势运行中的结构分歧、风险提示或止盈观察点。

### 3.4 30分钟信号不宜独立使用

30分钟信号样本数量最多，但单独使用时胜率和收益并不突出。初步判断：

- 日线更适合判断大结构；
- 60分钟更适合承担主要触发；
- 30分钟更适合做入场细化和风险位观察。

## 4. 核心研究问题

后续研究不应直接从“怎么赚钱”开始，而应先回答以下问题：

1. 一买、二买、三买、一卖、二卖在统计上分别意味着什么？
2. 哪些信号是趋势延续，哪些信号是左侧修复，哪些信号只是风险提示？
3. 信号在哪些市场环境下有效？
4. 信号在哪些个股状态下有效？
5. 多级别联立是否能显著改善裸信号表现？
6. 卖点到底适合作为清仓信号、减仓信号，还是风险监控信号？
7. 哪些统计稳定的结论可以转化为 V3 交易规则？

## 5. 研究课题设计

### 课题一：缠论信号含义研究

目标：判断一买、二买、三买、一卖、二卖分别代表什么市场行为。

核心假设：

- 一买可能是左侧修复信号，不适合直接右侧追买；
- 二买可能是结构修复后的确认信号；
- 三买可能是趋势延续信号；
- 一卖、二卖可能更多是风险提示，而非直接卖出信号。

验证方法：

- 分别统计各信号未来 1、3、5、10、20 根 K 线收益；
- 观察平均收益、胜率、中位数收益、收益分布偏度；
- 比较不同周期下同类信号的差异；
- 检查收益是否由少数股票贡献，避免被极端样本误导。

阶段产出：

- 《一买、二买、三买的统计含义报告》
- 《一卖、二卖是否能作为卖出信号报告》

### 课题二：周期分工研究

目标：确认日线、60分钟、30分钟在策略系统中的角色。

核心假设：

- 日线负责方向与结构背景；
- 60分钟负责交易触发；
- 30分钟负责入场细化，不适合独立决策。

验证方法：

- 对比单周期信号表现；
- 检验“日线不弱 + 60分钟买点”是否优于单独 60分钟买点；
- 检验“日线/60分钟条件满足 + 30分钟买点”是否优于单独 30分钟买点；
- 观察不同周期组合对收益、胜率、回撤、持有时间的影响。

阶段产出：

- 《多级别联立是否提升缠论信号有效性报告》

### 课题三：市场环境过滤研究

目标：判断缠论信号在哪些市场环境下更有效。

候选市场环境：

- 上升环境；
- 震荡环境；
- 下跌环境；
- 高波动环境；
- 低波动环境；
- 普涨环境；
- 分化环境。

核心假设：

- 上升环境中，趋势延续类信号可能更有效；
- 震荡环境中，结构修复类信号可能更有效；
- 下跌环境中，买点信号需要更严格过滤；
- 卖点信号在弱市中可能更适合作为风控信号。

验证方法：

- 以指数趋势、市场宽度、样本池均值收益划分市场环境；
- 分环境统计各信号未来收益；
- 对比同一信号在不同环境中的收益差异；
- 使用样本外区间验证环境分类是否稳定。

阶段产出：

- 《市场环境对缠论信号有效性的影响报告》

### 课题四：个股状态过滤研究

目标：判断同一个信号在哪类股票上更有效。

候选个股状态：

- 是否站上 MA20；
- 20日动量强弱；
- 60日动量强弱；
- 波动率高低；
- 距离20日高点/低点；
- 是否放量；
- 是否属于近期强势行业。

核心假设：

- 强势股上的二买、三买更可能延续；
- 弱势股上的一买可能只是反弹，不一定反转；
- 高波动状态下信号胜率可能下降，但收益弹性更高；
- 接近前高的三买和低位修复的三买可能是两种不同信号。

验证方法：

- 按个股趋势、动量、波动率、位置分组；
- 对比各组中同类信号的收益表现；
- 检查过滤条件是否提高胜率或收益风险比；
- 避免过度筛选导致样本过少。

阶段产出：

- 《个股状态过滤对缠论信号的影响报告》

### 课题五：卖点风控用途研究

目标：判断一卖、二卖如何用于持仓管理。

核心假设：

- 卖点不应直接等同于清仓；
- 强趋势中的卖点可能只是结构分歧；
- 弱环境中的卖点可能更适合作为减仓或止盈信号；
- 卖点需要结合跌破风险位、小级别转弱或均线破位确认。

验证方法：

- 比较卖点后未来收益是否显著低于无卖点状态；
- 按市场环境和个股趋势拆分卖点效果；
- 检验“卖点 + 跌破 MA20”“卖点 + 小级别转弱”等组合；
- 比较清仓、减仓、提高止损三种执行方式。

阶段产出：

- 《缠论卖点在持仓风险管理中的作用报告》

### 课题六：交易规则合成研究

目标：将稳定有效的统计结论转化为 V3 交易规则。

候选规则框架：

1. 市场环境不弱；
2. 日线结构不弱；
3. 60分钟出现一买、二买或三买中的优势信号；
4. 30分钟用于入场细化；
5. 风险位由结构低点、均线或波动止损共同决定；
6. 卖点先作为减仓观察，不直接清仓；
7. 跌破结构风险位或多周期转弱后退出。

验证方法：

- 先用简单规则回测；
- 再逐步加入过滤条件；
- 每加入一个条件都要回答：收益是否提升，回撤是否下降，交易次数是否还能接受；
- 最终做样本内、样本外和滚动窗口验证。

阶段产出：

- 《V3 缠论多级别交易规则设计与回测报告》

## 6. 研究优先级

建议按照以下顺序推进：

1. 完整 300 只事件级样本生成与市场环境分组；
2. 裸信号统计复核；
3. 周期分工研究；
4. 市场环境过滤研究；
5. 个股状态过滤研究；
6. 卖点风控用途研究；
7. V3 交易规则合成；
8. 样本外验证和实盘观察清单。

这个顺序的原因是：如果不先理解信号含义，就直接组合规则，很容易过拟合；如果不先完成环境和状态分组，就无法解释为什么同一个信号有时有效、有时无效。

## 7. 判断一个研究结论是否可用的标准

一个结论不能只看平均收益高不高。后续进入策略规则前，至少要满足：

1. 样本数量足够，不能只靠几十次事件；
2. 多数股票方向一致，不能只靠少数大涨股票贡献；
3. 平均收益、中位数收益、胜率不能互相矛盾太严重；
4. 在不同时间段表现不能完全相反；
5. 加入交易成本后仍有空间；
6. 规则简单、可解释、可执行；
7. 能明确失效条件和风险边界。

## 8. 当前阶段结论

当前全样本数据支持以下初步判断：

1. 缠论裸信号不适合直接机械交易；
2. 信号需要结合周期、市场环境、个股状态进行二次过滤；
3. 日线与60分钟信号更适合作为主要研究对象；
4. 30分钟信号更适合做入场优化；
5. 一卖、二卖不应直接视为清仓信号；
6. 后续真正有价值的研究问题是：在什么条件下，缠论信号从“结构标签”变成“交易优势”。

下一步建议先完成以下三件事：

1. 优化完整 300 只事件级分析脚本，生成完整 `event_samples.csv`；
